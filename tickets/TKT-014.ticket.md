# TKT-014: Align status/priority color conventions with tisket

## Problem
The tisket project has standardized color conventions:

**Priority colors:**
- urgent: `bg-red-500/10 text-red-600 dark:text-red-400 border-red-500/20`
- high: `bg-orange-500/10 text-orange-600 dark:text-orange-400`
- medium: `bg-blue-500/10 text-blue-600 dark:text-blue-400`
- low: `bg-slate-500/10 text-slate-600 dark:text-slate-400`

**Status colors:**
- open: `text-blue-500`
- in_progress: `text-yellow-500`
- resolved: `text-green-500`
- closed: `text-slate-400`

tisket-app should use these same conventions for visual consistency.

## Solution
Audit all status/priority indicators in tisket-app and align with these color patterns.

Create shared utility functions or component variants for consistency.
