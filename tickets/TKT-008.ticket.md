# TKT-008: Adopt OKLCH color space from tisket reference project

## Problem
The tisket reference project uses OKLCH color space for perceptually uniform colors. tisket-app may be using hex or HSL colors which don't provide the same perceptual consistency.

## Solution
Migrate CSS custom properties to OKLCH format matching tisket's approach.
